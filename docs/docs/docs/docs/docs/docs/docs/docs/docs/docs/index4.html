Citi Bike & Subway - A Preliminary Study

The start of Citi Bike riding seems to have a significant positive correlation with the clustering of subway stations on neighborhood level. However the result does not reveal the causal effect between the two variables, so further research is needed for a more solid result.

I used QGIS and GeoDa to study the spatial clustering effect of Citi Bike riding. The graph below shows the relationship between Citi Bike usage clustering and subway station presence. The red spots are high-high cluster, and the green ones are low-low cluster, while the white spots do not show significant clustering effect.
